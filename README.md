## 🔬 关于量子传感

量子传感是利用量子效应（如纠缠、叠加、相干）实现高灵敏度测量的技术。典型应用包括：
- NV色心磁力计
- 原子钟与惯性导航
- 量子陀螺仪
- 超导量子干涉仪（SQUID）
